# 🌿 Trailing Dreams – Plants and Things

## 🪴 Introducción
**Trailing Dreams – Plants and Things** es una plataforma web tipo e-commerce enfocada en la venta de plantas y productos para su cuidado. El diseño combina una estética natural con una experiencia de usuario clara, moderna y atractiva.

---

## 🎯 Objetivo
Desarrollar una tienda online escalable que permita:
- Explorar un catálogo de plantas  
- Visualizar información detallada de cada producto  
- Facilitar la compra mediante contacto directo por WhatsApp  

---

## 💡 Justificación
El interés por las plantas y los espacios naturales ha crecido significativamente, sin embargo, muchas tiendas no cuentan con plataformas digitales bien diseñadas. Este proyecto busca cubrir esa necesidad mediante una interfaz intuitiva, estética y funcional.

---

## 🛠️ Desarrollo del proyecto

### 📌 Descripción
El proyecto consiste en un e-commerce desarrollado con **HTML y CSS**, con una arquitectura preparada para escalar mediante JavaScript, permitiendo una interacción dinámica entre el usuario y los productos.

---

### ⚙️ Funcionalidades principales

- 🏠 **Home**
  - Presentación de la marca  
  - Banner principal  

- 🛒 **Shop**
  - Catálogo de productos  
  - Visualización de plantas  

- 🌱 **Product**
  - Vista detallada del producto seleccionado  
  - (Próximamente) Renderizado dinámico  

- 📞 **Contacto**
  - Información básica  
  - Acceso directo a WhatsApp  

- 💬 **Compra por WhatsApp**
  - Redirección directa para concretar la compra  

---
